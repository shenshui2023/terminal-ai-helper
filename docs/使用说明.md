# terminal-ai-helper 使用说明

terminal-ai-helper 是一个面向终端命令的桌面 AI 助手。它可以解释命令、补全命令、诊断报错、生成常用工具菜单，并支持本地 PowerShell、Windows 桌面面板、系统托盘、VS Code 右键和 SSH 远端会话。

## 快速开始

```powershell
cd E:\3.13-aliyun-codex\5.2\terminal-ai-helper
powershell -ExecutionPolicy Bypass -File .\scripts\install\all.ps1
. .\powershell\taih-profile.ps1
```

常用入口：

| 场景 | 推荐用法 |
| --- | --- |
| 解释本地当前命令 | `Alt+/` 或 `F2` |
| 打开管理面板 | `Alt+?`、`F3` 或 `taih-panel` |
| 本地智能补全 | `F4` 或 `Alt+P`；`Ctrl+Space` 只在没有被输入法占用时可用 |
| SSH 远端智能补全 | `F4` 或 `Alt+P` |
| SSH 远端控制面板 | `taih-ssh-panel` |
| 分析 SSH 标签页里已显示的文本 | 托盘全局热键 `Ctrl+Alt+/` 或 `Ctrl+Alt+F` |
| 常用工具菜单 | 管理面板选择 `tools（工具菜单）`，再选择工具集 |
| VS Code 选中文本 | 右键使用“终端 AI 助手” |

## 管理面板

```powershell
taih-panel
```

管理面板支持：

- `explain（解析命令）`：说明命令作用、常用参数、示例和风险。
- `fix（诊断报错）`：分析报错原因，给出修复步骤和验证命令。
- `complete（补全命令）`：根据当前输入生成候选命令。
- `tools（工具菜单）`：按工具集生成常用基础命令菜单。
- 配置模型、接口地址、超时时间、工具集、输出格式和自定义提示词。
- 查看历史记录、清缓存、复制输出。

## 智能补全

在 PowerShell 或 SSH 远端输入命令前缀，例如：

```powershell
kube get svc
```

按 `F4` 或 `Alt+P` 会弹出候选框。`Ctrl+Space` 在很多中文输入法里是中英文切换键，所以不作为主推荐键。

- 本地候选立即显示，`git`、`ssh`、`docker`、`npm`、`python`、`java`、`adb`、`kube/kubectl` 等常见命令不用先等 API。
- AI 候选会在后台追加，不要求所有命令都预存在本地。
- 候选可以在下方输入框直接修改，按 `Enter` 插回当前命令行。
- `解释` 按钮、`Ctrl+E` 或 `F1` 会把当前候选送到管理面板解释。
- `Esc` 关闭候选框。

## SSH 远端使用

目标是：你在 Windows Terminal 里 SSH 到服务器时，仍然在本机桌面弹出管理面板或补全候选框，API key 也只留在本机。

### 1. 本机启动 helper server

更新项目后建议使用 `--replace`，避免远端继续连到旧服务：

```powershell
node E:\3.13-aliyun-codex\5.2\terminal-ai-helper\bin\taih.js serve --port 17888 --replace
```

### 2. SSH 连接时打开反向隧道

```powershell
ssh -R 17888:127.0.0.1:17888 <用户名>@<主机>
```

例如：

```powershell
ssh -R 17888:127.0.0.1:17888 root@us-vpn
```

### 3. 远端加载 bash 集成

```bash
source /tmp/taih-bash.sh
```

如果服务器上还没有脚本，可以从本机传过去：

```powershell
scp E:\3.13-aliyun-codex\5.2\terminal-ai-helper\integrations\ssh\taih-bash.sh root@us-vpn:/tmp/taih-bash.sh
```

远端快捷键：

| 快捷键 | 功能 |
| --- | --- |
| `F4` 或 `Alt+P` | 打开本机桌面智能补全候选框，选择后写回远端当前命令行 |
| `Alt+/` | 解释远端当前命令行 |
| `Alt+F` | 诊断远端当前命令行 |
| `Alt+?` | 打开本机管理面板 |
| `Alt+T` | 打开本机工具菜单 |
| `Ctrl+Space` | 可选补全键，常被中文输入法抢占 |

远端命令：

```bash
taih explain 'systemctl status <服务名>'
taih complete 'kubectl get svc'
taih-complete-popup 'kubectl get svc'
taih-panel explain 'kubectl get svc -A'
journalctl -u <服务名> -n 80 --no-pager | taih fix
taih-tools
taih-tools linux k8s docker
```

`taih-tools linux k8s docker` 会设置当前远端会话的工具集，后续解释、补全和工具菜单都会优先围绕这些工具。

```powershell

```

远端链路可用反向隧道验证：

```powershell
node .\bin\taih.js serve --port 17888 --replace
ssh -R 17888:127.0.0.1:17888 root@us-vpn
```

```bash
curl -sS http://127.0.0.1:17888/health
taih-complete-popup 'kube get svc'
```

## 故障处理

快捷键没反应：

```powershell
taih-keys
taih-what-key
```

面板卡住：

```powershell
taih-panel-reset
taih-panel
```

API 不通：

```powershell
node E:\3.13-aliyun-codex\5.2\terminal-ai-helper\bin\taih.js doctor
```
