# terminal-ai-helper 使用说明

terminal-ai-helper 是一个面向终端命令的 AI 助手。它可以解释命令、补全命令、分析报错，并提供 PowerShell 快捷键、右侧管理面板、系统托盘、SSH 远端 shell、VS Code 右键等入口。

## 快速开始

进入项目目录：

```powershell
cd E:\3.13-aliyun-codex\5.2\terminal-ai-helper
```

检查配置：

```powershell
node .\bin\taih.js doctor
```

在当前 PowerShell 会话临时加载快捷键：

```powershell
. E:\3.13-aliyun-codex\5.2\terminal-ai-helper\powershell\taih-profile.ps1
```

一键安装本地入口：

```powershell
powershell -ExecutionPolicy Bypass -File E:\3.13-aliyun-codex\5.2\terminal-ai-helper\scripts\install\all.ps1
```

## 常用入口

| 场景 | 推荐入口 |
| --- | --- |
| 本地 PowerShell 当前命令 | `Alt+/`、`F4`、`Alt+F` |
| 打开右侧管理面板 | `Alt+?` 或 `taih-panel` |
| 本地智能补全 | `Ctrl+Space`、`Ctrl+Shift+Space`、`F4` |
| 像本地控制台一样操作 SSH 远端 | `taih-ssh-panel` |
| SSH 标签页里已经显示的文本 | 鼠标选中后按托盘全局热键 `Ctrl+Alt+/` 或 `Ctrl+Alt+F` |
| SSH 远端当前正在编辑的命令 | SSH 反向隧道 + `integrations/ssh/taih-bash.sh` |
| 常用基础命令菜单 | 面板里选择 `tools（工具菜单）`，再选择工具集 |
| VS Code 里选中的命令或日志 | 右键使用“终端 AI 助手” |

## 管理面板

打开面板：

```powershell
taih-panel
```

面板支持：

- `explain（解析命令）`：解释命令作用、参数、示例、风险，并补充相关命令。
- `fix（诊断报错）`：分析报错原因，给出修复步骤和验证命令。
- `complete（补全命令）`：根据当前输入生成候选命令。
- `tools（工具菜单）`：根据工具集生成常用命令菜单。

工具集可以在面板顶部选择或手动输入，例如：

```text
linux,ssh,systemd
k8s,kubectl,helm
docker,compose
python,pip,venv
embedded,serial,gcc,make
```

规则框可以写你的输出要求，例如：

```text
先说作用，再给相关命令和示例。
命令里的变量必须使用 <命名空间>、<服务名> 这类占位符。
每一项之间留空行，不要输出长表格。
```

## 命令行用法

```powershell
node .\bin\taih.js explain --style brief --tools k8s -- "kube get svc"
node .\bin\taih.js complete --json --no-cache --tools k8s -- "kube get svc"
node .\bin\taih.js tools --style brief --tools linux,k8s
node .\bin\taih.js config set tools linux,k8s,ssh
```

`complete` 不只依赖本地缓存。本地候选会先显示，AI 候选会后台追加；如果没有缓存，也会向中转站请求类似下面的候选：

```text
kube get svc -A
kube get svc -n <命名空间>
kube get svc -n <命名空间> -o wide
kube get svc <服务名> -n <命名空间> -o yaml
```

## SSH 文本和远端命令

### 本地 SSH 远端控制面板

如果你想在自己的桌面窗口里直接控制远端服务器，用：

```powershell
taih-ssh-panel
```

默认目标是 `root@us-vpn`。指定目标：

```powershell
powershell -ExecutionPolicy Bypass -File E:\3.13-aliyun-codex\5.2\terminal-ai-helper\powershell\ssh-panel.ps1 -Target root@43.112.115.205
```

面板左侧是远端控制区：上方选择 SSH 目标、Shell 和工具集，中间显示远端终端输出，下方输入命令并点击功能按钮。

面板右侧是 AI 输出和任务记录：可以解释当前命令、补全命令、诊断远端输出，也可以生成 Linux、K8s、Docker 等工具菜单。

这个方式不需要把 API key 放到服务器，也不需要在服务器安装项目。它由本机面板通过 SSH 执行远端命令，AI 请求仍然由本机发出。

验证远端连接：

```powershell
powershell -ExecutionPolicy Bypass -File E:\3.13-aliyun-codex\5.2\terminal-ai-helper\apps\powershell\ssh-panel.ps1 -SmokeTest -Target root@us-vpn
```

### 已显示文本的选区分析

本机 Windows Terminal 能看到 SSH 标签页里显示的文字，也能复制你用鼠标选中的内容。因此，如果你只是想分析屏幕上已经显示出来的远端命令、报错或日志，推荐用托盘全局选区：

```powershell
powershell -ExecutionPolicy Bypass -File E:\3.13-aliyun-codex\5.2\terminal-ai-helper\powershell\tray.ps1
```

用法：

- 在 Windows Terminal 的 SSH 标签页里选中一段远端文本。
- 按 `Ctrl+Alt+/`：解释选中文本。
- 按 `Ctrl+Alt+F`：诊断选中文本。

这个方式读取的是本机 Windows Terminal 选区/剪贴板，API key 仍然留在本机。

### 远端 bash/readline 快捷键

如果你想“不选中文本，直接读取远端当前正在编辑的命令行”，也可以使用远端 bash/readline 集成。原因是进入 SSH 后，当前输入已经交给 `ssh` 进程和远端 shell，本机 PowerShell 的 PSReadLine 不再管理这行命令。

本机启动 helper server：

```powershell
node E:\3.13-aliyun-codex\5.2\terminal-ai-helper\bin\taih.js serve --port 17888
```

使用反向隧道连接服务器：

```powershell
ssh -R 17888:127.0.0.1:17888 <用户名>@<主机>
```

远端加载 bash 集成：

```bash
source /path/to/terminal-ai-helper/integrations/ssh/taih-bash.sh
```

远端快捷键：

| 快捷键 | 功能 |
| --- | --- |
| `Alt+/` | 解释远端当前命令行 |
| `Ctrl+Space` | 补全远端当前命令行 |
| `Alt+F` | 诊断远端当前命令行 |
| `Alt+?` | 打开本机管理面板 |
| `Alt+T` | 打开本机工具菜单 |

远端命令：

```bash
taih explain 'systemctl status <服务名>'
taih complete 'kubectl get svc'
taih-panel explain 'kubectl get svc -A'
taih-tools
taih-tools linux k8s docker
journalctl -u <服务名> -n 80 --no-pager | taih fix
```

`taih-tools linux k8s docker` 会设置当前远端会话的工具集，后续解释、补全和工具菜单都会优先围绕这些工具。

## 测试

```powershell
npm run check
npm test
npm run test:api
```

`npm test` 会检查：

- Node.js 入口和模块语法。
- PowerShell 脚本能解析和加载。
- 快捷键和别名已注册。
- 本地补全候选可用。
- 补全候选框能在无 API 模式下返回本地候选。
- 管理面板子进程能启动并保持存活。

## 故障处理

面板没反应：

```powershell
taih-panel-reset
taih-panel
```

快捷键没反应：

```powershell
taih-keys
taih-what-key
```

API 不通：

```powershell
node E:\3.13-aliyun-codex\5.2\terminal-ai-helper\bin\taih.js doctor
```

清理缓存：

```powershell
node E:\3.13-aliyun-codex\5.2\terminal-ai-helper\bin\taih.js cache clear
```
